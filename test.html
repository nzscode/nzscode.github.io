Acess SQL

    Important Info :
  • To Access the psql server from GitBash first type: cd
  • Now enter the psql server by typing: psql -U postgres
  • The -U is for user
  • postgres, is the username
  • The password is whatever you have set up: make sure it is easy to remember.
  • As of 2023-10-07 , the password is password.

Using Python

import psycopg2
          
conn = psycopg2.connect(
    user='postgres',
    password='password',
    host='localhost',
    port=5432)

conn.autocommit = True
cur = conn.cursor()

# Actions to perform will go here:
cur.execute("""     """)

conn.commit()
conn.close()
