Copy Values

Important Info:

    Change the aqua code to any of the variations below replacing what is between < & > in italics with the appropriate names.
  • To insert values to specific tables in specific schemas, follow the <schema name.table name> convention.
  • When inserting column values from one column to another SQL considers it to be new entries and fills in the other columns with default values.
  • To Copy Values within the same table
    • INSERT INTO <table name>(<column name>) SELECT <column name> FROM <table name>;
  • To Copy Values from one table and insert it into another table (This is applicable even to tables in different schemas) 
    • INSERT INTO <table  name>(<column name>) SELECT <column name> FROM <table name>;


                            
        import psycopg2
        
        conn = psycopg2.connect(
        user='postgres',
        password='password',
        host='localhost',
        port=5432,
        database='demo_database')
        
        conn.autocommit = True
        cur = conn.cursor()
                          
        copy_values_same_table = """INSERT INTO demo_schema.demo_table(demo_col_2) SELECT demo_col_1 FROM demo_schema.demo_table;"""
                    
        cur.execute(copy_values_same_table)
        
        conn.commit()
        conn.close()
