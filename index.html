To convert a list to a CSV:
1. We have to import the Modules and Packages we need.
2. We must give our new pdf a name and point the path to the folder where we want to store our new pdf.
3. We must assign a variable for the Style Sheet Template we will use.
4. We can give our pdf a name, and a style.
5. We have to make a list where the data will be sorted and stored.
6. We need to assign table headers to a list and append it to the data list.
7. We need to assign a variable where we will declare the data that is to be the table.
8. We assign a variable to hold all our new style info.
9. We will set the header style to the table.
10. We must build the table.

import csv

f_names = [ "Sophia", "Liam", "Olivia", "Noah" ]
l_names = [ "Smith", "Kim", "García", "Müller" ]

from reportlab.platypus import SimpleDocTemplate, TableStyle
from reportlab.platypus import Paragraph, Spacer, Table, Image
from reportlab.lib.styles import getSampleStyleSheet
from reportlab.lib import colors

path = "../Resources/"
pdf = SimpleDocTemplate(path + "report.pdf")
styles = getSampleStyleSheet()

pdf_title = Paragraph("Names", styles["h1"])

full_names = []
table_headers = ['First Names', 'Last Names']
full_names.append(table_headers)
for i in range(len(f_names)):
    full_names.append([f_names[i], l_names[i]])

pdf_table = Table(data=full_names)

header_style = TableStyle([
    ('BACKGROUND', (0,0), (1,0), colors.grey),
    ('TEXTCOLOR', (0,0), (1,0), colors.blue)
])

pdf_table.setStyle(header_style)
pdf.build([pdf_title, pdf_table])

To see the new CSV file, we just have to navigate to the folder where we saved it and open the appropriate file.
