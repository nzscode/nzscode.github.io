import csv
path = "../Resources/"

data = [["a", 1], ("b", "f"), [3, "c"], [4, 5], (9, 8)]
with open(path+"demo.csv", 'w', newline='') as file:
    field_names = ['a', 'b']
    dict_writer = csv.DictWriter(file, fieldnames=field_names)

    dict_writer.writeheader()
    
    for i in range(len(data)):
        dict_writer.writerow({'a': data[i][0], 'b': data[i][1]})

    Important Info:

  • A Seperate config file can be made with the connection details to safeguard database access details.
  • When creating a table, you must always specify the columns and their types.
  • Serial can have the PRIMARY KEY attribute, which will help it when joining it with other tables for queries.
       The PRIMARY KEY makes each record unique.
  • NOT NULL => this field must have data in it, if there is no data, the table will not work.


      • The most often used Data Types in SQL tables.
  • Serial: 1 to 2Billion, consecutive auto incrementing number.
  • Integer: From -21,000 upto 21,000.
  • Decimal: Upto 13,000 before decimal, and 16,000 after decimal.
  • Money: From -92,000 upto 92,000.
  • Varchar(n): Variable length string where (n) is the user specified max length of characters.
  • Text: unlimited length of characters.
  • Bool: True / False.
  • Date: Saved in yy-mm-dd format.
  • Time(p)[without time zone]: Time of day no time zone.
  • Time(p): Time of day, including time zone.
  • Timestamp(p)[without time zone]: Date and time, without time zone.
  • Timestamp(p): Date and time, including time zone.
  • Json: Textual json data.
  • XML: XML data.
